# html-links-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What HTML tag is used to link to another website?

<a href="link">text content</a>

- What is an absolute URL?

Absolute URLs specify the complete address of the linked resource, including the protocol

- What is a relative URL?

Relative URLs only specify the path to the linked resource relative to the current page
a web address that specifies the location of a page or resource on a website relative to the current page
